# flomi - 项目档案

---

## 📱 项目概述

**项目名称：** Flomi 情绪记录  
**品牌语：** Tudo flui（万物皆流）  
**定位：** 在地图上定位你的情绪，获得即时的舒缓方案  
**特点：** 极简主义、隐私优先、游戏化交互

---

## 🎨 核心设计

### 色彩矩阵（四象限）

| 象限 | 情绪 | 色彩范围 |
|------|------|----------|
| 左下 | 沮丧 | #1a1a3e → #2d1b4e (深蓝紫 → 深紫) |
| 右下 | 焦虑 | #2d1b4e → #4a1a1a (深紫 → 深红) |
| 右上 | 喜悦 | #4a1a1a → #ff8c42 (深红 → 橙金) |
| 左上 | 平静 | #1a3e2e → #2d5a4a (深青绿 → 青绿) |

### 交互设计

- **长按 1 秒** → 标记情绪 + 震动反馈
- **实时色彩渐变** → 背景随坐标变化
- **舒缓方案弹出** → 分区提供 3 个选项

## 🧠 核心算法

### 坐标 → 情绪
```javascript
getMoodByCoordinate(x, y) → { mood, label, emoji, color }
// x: 能量值 (0-100)
// y: 愉悦度 (0-100)
```

### 颜色插值
```javascript
interpolateColor(color1, color2, factor) → hexColor
// RGB 线性插值
```

## 📊 数据结构

```javascript
{
  timestamp: 1711353000000,   // 时间戳
  x: 72,                      // 能量值 (0-100)
  y: 58,                      // 愉悦度 (0-100)
  mood_label: "喜悦",         // 中文标签
  mood_type: "excited",       // 英文类型

```

**存储：** 完全本地 (wx.getStorage)，最多 30 条，自动滚动

## 💡 可应用的想法

1. **情绪分析伦理**：在 AI 分析用户情绪时，需要设置边界，避免过度解读或给出医疗建议
2. **产品温度**：界面设计要有"人情味"，让用户感受到被理解和关怀
3. **主动帮助**：不只是被动记录情绪，可以主动询问用户是否需要帮助
4. **内容可信度**：考虑与心理健康专家合作，提供可信的内容资源
